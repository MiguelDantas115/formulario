Perguntas e Respostas:

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"? O atributo "placeholder" exibe um texto auxiliar no campo até que o usuário comece a digitar.

  2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"? O atributo "checked" marca o item como selecionado inicialmente.

  3. Na tag <input> para que serve o atributo "value"? Define o valor enviado para o servidor ao enviar o formulário.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez? O atributo "name" deve ser o mesmo para os botões que formam o grupo.

  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag? A tag <label> associa texto descritivo a um campo de entrada, e o "for" liga o rótulo ao campo pelo ID.

  6. Na tag <option> para que serve o atributo "selected"? Indica a opção que deve aparecer selecionada inicialmente.

  7. Nas tags de formulário para que serve o atributo "required"? Obriga o preenchimento do campo antes de enviar o formulário.

  8. Para que serve a tag <textarea>? Permite a entrada de texto em várias linhas.

  9. Na tag <form> para que serve o atributo "action"? Define o URL para onde os dados do formulário serão enviados.

  10. Na tag <form> para que serve o atributo "method"? Especifica o método de envio do formulário (GET ou POST).

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>? O valor deve ser "POST".

  12. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos? Deve-se utilizar o "POST".

  13. O que as tags <fieldset> e <legend> produzem? Agrupam campos do formulário com uma legenda descritiva.
